# Crawler

## Install Node.js

### Windows

Install Node.js: http://nodejs.org/download/

### Other

```
apt-get install python git build-essential
git clone git://github.com/joyent/node.git
cd node
git checkout v0.10.4
./configure
make -j2
make install
```

## Install crawler

```
npm install
```

## Run

```
npm start
```
